\f0\fs24 \cf0 # \uc0\u55357 \u56538  Book Catalog Web Scraper\
\
A Python tool that scrapes book titles, prices, and ratings from all 50 pages of the 'Books to Scrape' website.\
\
## \uc0\u55357 \u57056  Skills Demonstrated\
* **Pagination:** Navigating through 50+ web pages automatically.\
* **Data Cleaning:** Converting string-based ratings and currency symbols into clean data.\
* **File I/O:** Exporting data into CSV format for analysis.\
\
## \uc0\u55357 \u56960  How to Run\
1. Install dependencies: `pip install -r requirements.txt`\
2. Run the script: `python scraper.py`\
